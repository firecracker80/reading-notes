# Summary of Relatinal Databases and SQL

> A relational database is a collection of data that arranges data in preset relationships. The data is stored in one or more tables arranged in columns and rows, making it simple to view and comprehend how various data formats connect to one another. Structured Query Language (SQL), is a query language used to contact databases. It's the preferred language for relational database management systems, according to ANSI (American National Standards Institute).
